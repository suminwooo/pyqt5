※ 우선적으로 요청사항에 따라 개발은 해두었습니다. 아직 모델이 완성단계가 아니기 때문에 원리와 개선점 혹은 필요한 사항을 전달해 주시기 바랍니다.

1. 폴더를 열면 많은 폴더와 파일이 있으나 중간에 exe파일인 pyqt5_model 을 실행
2. 파일을 실행하면 저번에 수정했던 UI로 보여짐

3. TRACK1
	3.1. 패턴 인식 기능 : 특정폴더에 csv 파일 떨구기 -> 전체 폴더를 보면 data파일이 존재(내부는 수정예정) -> sample_data 내부에 5개의 csv 파일이 존재 -> 필요한 파일을 네모 박스에 끌어 넣기 -> 드래그 된 파일을 누르고 아래의 파일선택 버튼 클릭(필수) -> 이후 머신러닝 test_data_input 버튼을 누르면 아래에 각 유형별 % 표시 -> 이후 아래의 전치리 결과 Excel 출력 버튼을 누르면 data폴더내에 새로운 csv 파일이 생성됨
	+ 사용되는 모델의 경우에는 data폴더내에 model_data & scaler_date 를 활용하여 결과가 나옴

	3.2 모델 업데이트 기능 : 독자적으로 구현되어 있는 버튼. data폴더 내에 update_data에 데이터가 들어 있을때 버튼을 누르면 내부의 데이터를 활용해 새로운 모델을 만듬. data파일 내에 new_model.sav & new_scaler.sav 가 생성됨. -> 이후 이를 이용하고 싶으면 이 파일의 이름을 using_model.sav & using_scaler.sav로 변경하여 3.1.의 model_data & scaler_data에 넣어서 사용하면됨(기존의 모델은 이름 변경 필수) 
	※ 모델의 경우 시간이 많이 걸리며 어플리케이션이 응답하지 않다고 될 수 있음. 모델링이 완료되면 정상적으로 돌아옴.(샘플의 경우 3-5분 정도 소요)

4. TRACK2 &3
	4.1 그래프 출력 기능 : 특정 폴더에 csv 파일 떨구기에 excel 파일을 추가시켜주고 파일을 선택한뒤 선택 버튼을 눌러줌 -> 이후 축을 선택(2차원 or 3차원으로 선택이 가능하며 현재의 데이터를 사용하지 않고 가상의 데이터를 활용한점 참고, Z축을 none으로 하면 2차원으로 표현됨) -> 이후 그래프 출력버튼을 누르면 그래프가 표현됨.
	
	4.2 그래프 상세 검색 : 확인한 그래프를 통해 x,y,z축의 범위를 설정해준뒤 좌표 범위 셋팅 버튼을 누르고 좌표 그래프 출력 버튼을 누름.-> 팝업창에서 버튼을 눌러서 기존에 셋팅한 좌표 확인.-> 이후는 아직 구현 하지 않음.
	※ 팝업창은 동시에 여러개 띄울 수 있음. 좌표축을 새로 입력하고 버튼을 두개 누를 경우 여러가지 범위로 출력하여 볼 수 있음.


○ 이전에 회의한 결과로 데모버전으로 프로그램을 만들었습니다. 기능이 생각하시는것 처럼 구현이 잘 되어 있을지는 모르겠지만, 모델을 만들면서 계속 수정해 나갈 예정입니다. 추가적으로 필요한 부분이나 수정사항이 있으면 요청 부탁드리겠습니다.
